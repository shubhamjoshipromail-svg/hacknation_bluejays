# Team Agent Rules

- Read README.md before starting.
- Never work directly on main.
- Create a separate branch for every task.
- Work only on the task assigned by the user.
- Do not delete, rewrite, or refactor unrelated teammate work.
- Do not commit API keys, secrets, or .env files.
- Test the change before finishing.
- Push the feature branch and open a Pull Request into main.
- Do not merge the Pull Request.
- Report the branch, files changed, tests run, and any conflicts.
